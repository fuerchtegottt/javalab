# JPCloudConnector
![LOGO](res/logo.png)

# description 
JAVA based api calls for PCloud

## purpose
trigger useful api calls on your PCloud account

## how to..
1. launch app JPCloudUI
2. enter App ID + Secret
3. push OAUTH Button
4. login in browser window
5. token into UI
6. push connect
7. select API function from dropdown list
8. push execute
9. check results in console area

# requirements
 - PCloud account
 - PCloud API Key (Application ID + Secret)
 - jre > 1.8 installed
