# Description 
html/xml based lightweight blog 

## purpose
pattern can  be used to create a simple blog based on an xml data source
	
## features
* XSLT - transformation (XML -> html blog)
* blog entries with link, image, description and source code block
	
## technology
* html
* xml
* xslt
* css
